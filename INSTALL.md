# Enterprise Security System

## One-Command Installation

### Windows
```bash
git clone https://github.com/yuhisern7/enterprise-security.git
cd enterprise-security
setup.bat
```

### Linux/Mac
```bash
git clone https://github.com/yuhisern7/enterprise-security.git
cd enterprise-security
chmod +x setup.sh && ./setup.sh
```

**Dashboard:** http://localhost:5000
